QA Java Project

Реализован сервис авторизации и аутентификации пользователей на базе JWT-токенов с контролем доступа по ролям (RBAC). Поддерживается регистрация, логин/логаут, управление пользователями и сессиями.
​
Ключевые возможности
API-эндпоинты для CRUD пользователей.

Миграции БД для схемы таблиц.

Автоматические тесты и линтинг кода.

Локальный запуск через Docker Compose.

Инструменты и стек
Язык: Python.

Фреймворк: Poetry (управление зависимостями), Peewee (ORM).

БД: PostgreSQL.

Контейнеры: Docker, docker-compose.

Тестирование/качество: Pytest, Ruff (линтер), MyPy (типизация), pre-commit хуки.

Сборка/запуск: Makefile команды (tests, lint, build, run).
